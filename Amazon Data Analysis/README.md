# Crash Data

This project cleans up a dataset from Kaggle. The data involved sales data from Amazon sales and, among other things, converted the price from Pounds to US Dollars.

## Installation

The program was developed in Jupyter Notebook and was written in Python 3. 

## Usage

Run the code. Any required packages are imported at the beginning.
